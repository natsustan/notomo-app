Changelog

Release notes for Notomo.

v0.4.10

  • Added a built-in Skills library so you can quickly set up common actions like Improve Writing, Proofread, Explain, and Reformat.
  • Improved Skills setup in Settings with clearer folder creation, add feedback, and refresh behavior.
  • Renamed the selection launcher wording to Actions, making the panel clearer when launched from selected text or PopClip.
  • Improved ChatGPT follow-up context when continuing an Inline AI or Actions result in the assistant.
  • Hid the inline formatting toolbar when only an Apple Notes attachment is selected.
  • Refined Settings organization, provider status labels, translations, and app launch handling.

v0.4.9

  • AI now connects through ChatGPT sign-in, making setup clearer and removing the old local helper dependency.
  • Added a menu bar entry for opening the Skills Launcher directly.
  • Improved keyboard handling around Apple Notes, slash triggers, and Notomo panels to reduce stalls while typing.
  • Made Inline AI status and loading panels more compact, stable, and better fitted to their content.
  • Softened the AI assistant panel visuals with calmer glass surfaces.

v0.4.8

  • Added PopClip support for sending selected text directly into the Skills Launcher.
  • Added menu bar shortcut hints so it is easier to see how to open the AI assistant.
  • Improved Inline AI copy behavior, including clearer handling when a result has no changes to apply.
  • Made Inline AI result previews better at recognizing unchanged output, so Notomo avoids offering confusing apply actions.

v0.4.7

  • Added a global Skills Launcher for using your saved Skills on selected text outside Apple Notes.
  • Skills results can now replace the original selection or insert text below it when Notomo can safely return to the source app; Copy and Continue in chat remain available.
  • Improved launcher and inline toolbar placement so panels appear more reliably near the current text selection, including after mouse selections and double-clicks.
  • Made the Skills Launcher dismiss cleanly when clicking outside it.
  • Made Codex-based AI actions feel faster by reusing warm sessions and preparing Inline AI and assistant model choices earlier.
  • Improved background AI session cleanup and Gemini CLI handling for more stable headless runs.

v0.4.6

  • Added a command palette for Apple Notes, with quick access to Quick Open, AI Assistant, Settings, and note export actions.
  • Added note export actions for copying the selected note as Markdown or plain text, copying its Apple Notes link, and saving it as Markdown or PDF.
  • Improved Apple Notes export parsing so copied and exported notes keep cleaner headings, lists, quotes, code blocks, and filenames.
  • Made Inline AI safer when applying results: Notomo now checks that Apple Notes is still focused before replacing or inserting text.
  • Improved Quick Open and Settings readability, including clearer loading states, section descriptions, sidebar icons, and a configurable command palette shortcut.
  • Made synced Skills available from a saved cache across app launches, so they remain visible even before Apple Notes finishes refreshing.

v0.4.5

  • Added Templates support: create a top-level Apple Notes folder named "Templates" or "templates", add one note per template, then insert those templates from the slash command panel.
  • Added template variables for the current date and time, so reusable notes can automatically include today's date or the current time when inserted.
  • Added /date and /time slash commands for quickly inserting the current date or time into Apple Notes.
  • Added date and time format settings for both slash commands and template variables, with localized defaults for each app language.
  • Improved template and Skills syncing with cached results, clearer refresh state, and more reliable access to Apple Notes content.

v0.4.4

  • Fixed inline formatting from the floating toolbar so the Apple Notes editor is refocused before applying the change.
  • Improved toolbar behavior after choosing an inline formatting action, reducing cases where focus stayed on Notomo instead of returning to the note.
  • Reset the Settings window's initial focus each time it opens, so keyboard focus no longer lands on a stale control from the previous visit.

v0.4.3

  • Added Google Gemini API support, so Notomo can use Gemini models directly with an API key instead of requiring the Gemini CLI.
  • Added Cloudflare Workers AI support, including account/token setup, credential checking, and model selection from Settings.
  • Reworked the AI model picker to group models by provider and connection type, making it clearer whether a model runs through a CLI or an API connection.
  • Improved AI chat title generation and inline AI result previews when using the newer provider options.
  • Made the Settings window resizable while keeping each settings page readable at wider sizes.

v0.4.2

  • Added optional Vim-style navigation for Quick Open, so you can move through results, open a note, or close the panel with Control-H/J/K/L.
  • Improved Skills sync from Apple Notes, with faster refreshes, shared Notes database access, clearer folder status, and better retry behavior after refresh errors.
  • Preserved more formatting when reading Skills from Apple Notes, including Markdown, ordered lists, underscores, and short Chinese/Japanese/Korean instruction lines.
  • Refined Settings with clearer sections, improved help text, updated icons, and fewer unavailable controls.
  • Improved translations and wording across the assistant panel, Quick Open, slash commands, formatting actions, and appearance settings.

v0.4.1

  • Added custom AI instructions, so you can tell Notomo how you want chat replies and inline edits to be shaped.
  • Improved inline AI results so replacement text streams in more reliably and stays focused on the selected text.
  • Refined AI settings, including clearer provider controls, memory options, and chat history handling.
  • Improved Codex and Gemini command detection for Macs using Node through nvm, fnm, Volta, asdf, Bun, npm-global, or mise.
  • Made Notomo prefer newer version-managed Node installs when several are available.

v0.4.0

  • Added inline AI, so you can select text in Apple Notes and ask Notomo to improve, rewrite, summarize, or respond with your own prompt.
  • Added customizable actions to the inline toolbar, with clearer icons for common writing and formatting tasks.
  • Added quick list commands for Apple Notes, including bulleted lists, numbered lists, and checklists.
  • Improved the AI assistant panel with calmer colors and a more polished reading experience.
  • Made note context handling more reliable, so Notomo keeps note titles intact while refreshing lightweight AI context.
  • Improved Apple Notes focus handling, especially around Markdown shortcuts and deleting trigger text.
  • Refined Quick Open visuals and handling while typing.
  • Simplified the menu bar experience so the most useful actions are easier to find.

v0.3.0

  • Improved AI note context refresh when opening the assistant panel.
  • Improved Quick Open behavior while using marked text or input method composition.
  • Made Quick Open ignore recently deleted notes.
  • Improved the Quick Open shortcut flow and visual details.
  • Added an accessibility permission prompt so first-time setup is clearer.
  • Improved note context mentions in the assistant picker.
  • Polished several small details across the assistant, Quick Open, and settings.

v0.2.0

  • Added Simplified Chinese localization and an app language setting.
  • Added Quick Open for finding and opening Apple Notes more quickly.
  • Added an inline formatting toolbar for selected text in Apple Notes.
  • Added rich text copy options in the AI assistant panel.
  • Improved AI conversations when working with images from your notes.
  • Added a skill mention picker and skill management for the AI assistant.
  • Refreshed the app icon, menu bar icon, slash command icons, and assistant visuals.
  • Improved chat behavior while replies are streaming, including pinned sent messages and clearer thinking states.
  • Made update settings easier to understand.

v0.1.0

  • Initial Notomo beta for writing faster in Apple Notes.
  • Added Markdown-style shortcuts for headings, lists, quotes, code blocks, dividers, and note links.
  • Added the floating command panel for inserting common Apple Notes formatting without leaving the keyboard.
  • Added the AI assistant panel with streaming replies, chat history, and note context.
  • Added AI provider settings so you can choose the assistant setup that works best for you.
  • Added note image handling so images from Apple Notes can be included when asking AI.
  • Added basic settings for app behavior, permissions, appearance, and chat history.
