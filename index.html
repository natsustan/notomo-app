Notomo brings Markdown shortcuts and AI assistant to Apple Notes.

Use slash commands and note-aware AI beside the note you are already editing.

Download for macOS

v0.3.0 · Requires macOS 15 or newer

Works beside Apple Notes

Notomo appears when Notes is active and stays out of the way when it is not.

Turns typing into commands

Use Markdown triggers for headings, checklists, code-style text, quotes, and note-link helpers.

Adds local AI workflows

Ask an assistant to use selected text, the current line, visible note text, or the full note as context.

Markdown shortcuts

Write Notes with familiar text triggers.

Turn plain typing into Apple Notes formatting without leaving the keyboard.

  • Titles, headings, checklists, monostyled text, and block quotes.
  • CJK-friendly aliases for punctuation that is easier to type.
  • Note-link helpers for connecting notes quickly.

Command layer

Open formatting and AI actions in place.

A slash menu and inline toolbar bring common actions near the note you are editing.

  • Keyboard-first command menu for formatting, tables, and AI chat.
  • Inline toolbar for quick Apple Notes styling.
  • Menu bar access when you need Notomo outside the editor.

Note-aware AI

Ask with the current note already in view.

Use selected text, the current line, visible text, the full note, and recovered inline images when they are available.

  • Works with installed Codex or Gemini command line tools.
  • Keeps unavailable models out of the picker.
  • Saves chat history with retained note image attachments.

Your notes stay in Apple Notes.

Notomo observes and assists from the outside. It does not replace Apple Notes or store notes in its own database.
