Notomo brings Markdown shortcuts and AI assistant to Apple Notes.

Use slash commands and note-aware AI beside the note you are already editing. Your notes stay in Apple Notes.

Download for macOS

v0.3.0 · Requires macOS 15 or newer

Works beside Apple Notes

Notomo appears when Notes is active and stays out of the way when it is not.

Turns typing into commands

Use Markdown triggers for headings, checklists, code-style text, quotes, and note-link helpers.

Adds local AI workflows

Ask an assistant to use selected text, the current line, visible note text, or the full note as context.

Notes Workflow

Markdown triggers
— Type familiar shortcuts for titles, headings, checklists, monostyled text, block quotes, and CJK-friendly aliases.
Slash commands
— Open formatting actions, table insertion, and AI chat from a keyboard-first command panel.
Inline toolbar
— Apply common Apple Notes formatting near the current editing context.

AI Context

Note-aware chat
— Use selected text, the current line, visible note text, full note content, and recovered inline images when available.
Local providers
— Work with installed Codex or Gemini command line tools, with unavailable models hidden from the picker.
Chat history
— Keep past AI conversations and retained note image attachments.

Mac Native

Menu bar access
— Quick open from the menu bar or keyboard-focused Apple Notes workflows.
Settings
— Control launch at login, menu bar visibility, appearance, language, feature preferences, permissions, and AI providers.
Sparkle updates
— Online update support through the public appcast feed.

Your notes stay in Apple Notes.

Notomo observes and assists from the outside. It does not replace Apple Notes or store notes in its own database.
